Chceš vědět víc, podívej se na Czechitas a přihlaš se na kurz!

DevTools

Díky DevTools mohu zjistit lépe, proč něco na webové stránce nefunguje

  • v záložce Elements si mohu zobrazit DOM strukturu stránky, případně CSS styly
  • v záložce Network vidím http metody, které byly zavolány a jejich statusy
  • v záložce Application se můžu podívat např. na Cookies, nebo Cache

Postman

Jak získat zdrojový kód HTML stránky

  • otevřu Postmana
  • otevřu novou záložku
  • nastavím http metodu GET
  • do řádku vedle napíšu URL adresu stránky
  • kliknu na tlačítko Send

XML vs. JSON

Oba formáty jsou určeny pro přenos dat a jsou nezávislé na počítačové platfomě

XML = Extensible Markup Language

  • musí mít právě jeden kořenový element (root)
  • elementy musí mít startovací a ukončovací značku
  • stromova struktura, tj. každý elememnt může obsahovat další elementz

JSON = JavaScript Object Notation

  • mají strukturu klíč: hodnota
  • nelze pro nej udelat validacni strukturu (narozdil od XML)
